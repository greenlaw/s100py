S1xxWritesOwnGroupBase¶

class s100py.s1xx.S1xxWritesOwnGroupBase(recursively_create_children=False, **kywrds)¶

Bases: s100py.s1xx.S1xxAttributesBase

Derive things that either create a dataset and have to combine data into it or make multiple sub groups that the parent can’t predict The S1xxAttributesBase will call the derived class’ writer without pre-making group for it. i.e. the derived class can specify its own group name or dataset and apply specialized logic as needed.
