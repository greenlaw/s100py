GeographicExtent¶

class s100py.s100.GeographicExtent(recursively_create_children=False, **kywrds)¶

Bases: s100py.s1xx.S1xxAttributesBase

4.2.1.1.12 of v2.0.0 The class EX_GeographicExtent is a metadata class from ISO 19115. It is a component of the metaclass EX_Extent. The use of EX_GeographicExtent is optional. When used it describes the spatial boundaries of the Tracking List elements within the bounds established by CV_GridEnvelope for the BathymetryCoverage. That is, the tracking list may carry information corresponding only to a portion of the spatial extent covered by the BathymetryCoverage. There is one attribute and one subtype.

Attributes Summary

extent_type_code

The attribute extentTypeCode is a Boolean value.

Attributes Documentation

extent_type_code¶

The attribute extentTypeCode is a Boolean value. It is used to indicate whether the bounding polygon/box encompasses an area covered by the data or an area where data is not present. In S-102 it is set to 1.
