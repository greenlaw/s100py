FeatureInformationDataset¶

class s100py.s100.FeatureInformationDataset(*args, **opts)¶

Bases: s100py.s100.Chunking, s100py.s1xx.S1xxDatasetBase, abc.ABC

This class comes from S100 – 10c-9.5 Feature information group. This class serves to keep a list of FeatureInformation objects which will be turned into a compound array of strings in the HDF5 file.

The metadata_name property must be overridden. The metadata_type will likely be overridden with a specific subclass for the s100+ spec
